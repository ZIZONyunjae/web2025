alert() 함수
* 알림 창 표시
* alrert() 함수의 괄호 안에 메시지를 입력하거나 변수를 사용 > 알림 창에 텍스트나 변숫값 표시

confirm()함수

확인 창 표시
*확인 창 표시(컨펌 창이라고도 함)
*[확인] 버튼과 [취소] 버튼이 있어서 사용자가 어떤 버튼을 클릭했는가에 따라 다르게 동작하도록 할수있다.
ex)confirm(내용) = confirm("종료하시겠습니까"?)

prompt() 함수

*프롬프트 창 - 사용자가 간단한 값을 입력할 수 있는 창 표시
*프로그램 실행에 필요한 값을 입력할 수 있는 창 표시
*기본 값을 지정하지 않으면 텍스트 필드가 빈 상태로 표시됨
ex)prompt(내용) 또는 prompt(내용, 기본값)
