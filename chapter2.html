confirm()함수

확인 창 표시
*확인 창 표시(컨펌 창이라고도 함)
*[확인] 버튼과 [취소] 버튼이 있어서 사용자가 어떤 버튼을 클릭했는가에 따라 다르게 동작하도록 할수있다.
ex)confirm(내용) = confirm("종료하시겠습니까"?)

prompt() 함수

*프롬프트 창 - 사용자가 간단한 값을 입력할 수 있는 창 표시
*프로그램 실행에 필요한 값을 입력할 수 있는 창 표시
*기본 값을 지정하지 않으면 텍스트 필드가 빈 상태로 표시됨
ex)prompt(내용) 또는 prompt(내용, 기본값)

사용자가 많이 입력할 것 같은 값을 기본 값으로
ex) prompt("컨퍼런스 참석 여부(예/아니오)","예")

프롬프트 창에서 입력하지 않고[취소]를 누르면? > 결과값 null

confole.log()함수

*콘솔 창에 괄호 안의 내용을 표시함
*자바스크립트 소스를 작성하면서 중간중간에 프로그램이 제대로 동작하는지 확인하는 용도로 자주 사용
*콘솔 창에 결과를 표시하는 함수는 많지만 주로 console.log()를 많이 사용함
*괄호 안에 텍스트나 변수를 사용할 수 있음
ex) console.log(내용)

document.write()함수

*괄호 안의 내용을 웹 브라우저 화면에 표시함
*실제 웹 브라우저 화면에 내용을 표시할 때에는 DOM을 이용하지만, 아직 DOM을 공부하지 않았기 때문에 일단 document.write()함수 사용.
*document.write()문에서 연결 연산자(+)를 사용할 수도 있고, 템플릿 리터럴을 사용할 수도 있다. ex) document.write(내용)

*값을 저장해 놓은 메모리 공간에 문패를 붙이는것과 같다.
*변수 이름을 쉽게 가져와서 그 안의 값을 사용할 수도 있고, 같은 위치에 바뀐 값을 저장할 수도 있음
