prompt() 함수

*프롬프트 창 - 사용자가 간단한 값을 입력할 수 있는 창 표시
*프로그램 실행에 필요한 값을 입력할 수 있는 창 표시
*기본 값을 지정하지 않으면 텍스트 필드가 빈 상태로 표시됨
ex)prompt(내용) 또는 prompt(내용, 기본값)

사용자가 많이 입력할 것 같은 값을 기본 값으로
ex) prompt("컨퍼런스 참석 여부(예/아니오)","예")

프롬프트 창에서 입력하지 않고[취소]를 누르면? > 결과값 null

ex) console.log(내용)

document.write()함수

*괄호 안의 내용을 웹 브라우저 화면에 표시함
*실제 웹 브라우저 화면에 내용을 표시할 때에는 DOM을 이용하지만, 아직 DOM을 공부하지 않았기 때문에 일단 document.write()함수 사용.
*document.write()문에서 연결 연산자(+)를 사용할 수도 있고, 템플릿 리터럴을 사용할 수도 있다. ex) document.write(내용)

변수에대해서

변수: 프로그램에서 사요ㅕㅇ하기 위한 값을 담아놓는 바구니

변수 이름 지정하기

*값을 저장해 놓은 메모리 공간에 문패를 붙이는것과 같다.
*변수 이름을 쉽게 가져와서 그 안의 값을 사용할 수도 있고, 같은 위치에 바뀐 값을 저장할 수도 있음
>따라서 변수 이름은 서로 다르게 만들어야 함

변수 이름 정하는 규칙

1) 변수 이름은 숫자로 시작할 수 없고 이름 안에 공백이 포함되어 있으면 안된다.
>>> ex) current, _current, $currnet //사용가능

2)자바스크립트는 영문자의 대소문자를 구별한다.
>>> ex) current,Current, CURRENT //모두 다른변수

3)한 단어로 이루어진 변수를 사용할 때에는 주로 소문자 사용
>>>currnet, age, sum

4)두 단어 이상으로 이루어진 변수는 언더바(_)로 연결하거나 중간에 대문자를 섞어 사용
>>> ex) currnet_year, total_area //스네이크 표기법, currentYear, totalArea //카멜 표기법

5)자바스크립트에서 미리 정해 놓은 예약어(let등) 변수 이름으로 사용할 수 없음

6)무의미한 변수 이름은 피한다.

변수 선언 및 할당

1) 변수 선언: 키워드 let이나 const다음에 변수 이름을 적어서 변수를 선언한다.
ex) let 변수명; const변수명; const는상수를 위한 예약어. 변수에 담아놓고 사용함
사용 예제 >>>
let result;
result = 10;

2)변수에 값 할당: 번수 오른쪽에 = 기호를 붙이고 오른쪽에 저장할 값이나 식을 작성한다.
ex) 변수명 = 값 또는 식;
3) 변수 선업과 값 할당을 동시에 할 수도 있다.
let 변수명 = 값 또는 식;

자료형 data type이란

*프로그램에서 처리하는 자료의 형태
*자료형, 자료유형, 데이터 타입 등으로 부름
*자바스크립트 자료형은 크게 '원시형'과 '객체'로 나눔
>>> 원시형(promitive type): 하나의 값만 가지고 있는 자료형
>>> 객체(object): 원시형 외의 모든 자료
