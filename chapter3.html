산술 연사자-증감연산자

a = 10
sum = a + 5 //15
a //10

sum = a++ +5 //15
a //11

증감 연산자는 어느 위치에 붙느냐에 따라처리 방법이 달라짐
a = 10
a++ //10
a //11

a = 10
++a //11
a //11

x = 10, y = 4, result
result = x + y-- //14
y //3

result = ++x-y //8
x //11

할당 연산자(대입 연산자)

대입 연산자 표
종류 설명 예시
= 연산자 오른쪽의 값을 왼쪽 변수에 할당합니다. y = x + 3
+= y = y + x를 의미합니다. y += x
-= y = y - x를 의미합니다. y -= x

/= y = y / x를 의미합니다. y /= x
%= y = y % x를 의미합니다. y %= x

비교 연산자

피연산자 2개의 값을 비교해서 true나 false로 결괏값 반환
비교 연산자는 조건을 확인 할 때 자주 사용하는 연산자이다.
비교 연산자 설명
== , != 두 개의 값이 같은지, 같지 않은지 확인
< , <= 왼쪽 값이 오른쪽 값보다 작은지 혹은 작거나 같은지 확인
> , >= 왼쪽 값이 오른쪽 값보다 큰지 혹은 크거나 같은지 확인
=== , !== 두 개의 값이 자료형까지 완벽하게 같은지, 같지 않은지 확인

== 연산자 와 !=연산자

피연산자의 자료형을 자동으로 변환해서 비교 3 == "3" //true
3 !=="3" //false

3 === "3" //false
3 !== "3" //true

문자열의 비교

피연산자가 문자열이라면 문자열에 있는 문자들의 아스키값을 비교해서 결정한다.
대략적 아스키값 순서: 제어문자 < 특수 기호 < 숫자 < 영대문자 < 영소문자

"A" > "B" //false("B"의 아스키값이 더 크므로)
"A" < "a" //ture(영소문자의 아스키값이 숫자보다 크므로)
글자가 여러 개인 문자열을 비교할 경우 맨 앞의 문자부터 하나씩 비교한다.
"Javascript" < "JavaScript" //false(소문자 아스키값 > 대문자 아스키값) 영대문자보다 소문자가 더큼

논리 연산자

불리언 연산자, true,false를 처리하는 연산자
논리 연산자 설명 표
종류 기호 설명
OR 연산자 || 피연산자 중 하나만 true여도 true가 됩니다.
AND 연산자 && 피연산자가 모두 true일 경우에만 true가 됩니다.
NOT 연산자 ! 피연산자의 반댓값을 지정합니다.

if ... else 문을 사용해서 3의 배수일 때와 아닐 때 할 일을 분리
결과는 어떻게 표시할까? – alert() 문 (이외에 document.write()나 console.log()도 가능)
